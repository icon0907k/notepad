**선형 자료구조** 는 데이터들이 순서대로 나열되어 하나의 데이터 뒤에 하나의 데이터가 연결되는 형태의 자료구조를 의미한다.

**선형 자료구조의 특징**
	1. 순서성 : 데이터들이 순서대로 나열되어 있기 때문에, 각 데이터의 위치를 인덱스나 순서로 쉽게 파악한다. 
	2. 1 : 1 관계 : 데이터는 앞 뒤 데이터와 하나씩 연결되어 있다.
	3. 접근 방식 : 데이터에 접근하기 위해서 일반적으로 처음부터 순차적으로 이동한다. 

선형 자료구죠 종류 
	1.	배열
	2.	리스트
	3.	스택 후입선출(LIFO) 구조
	4.	큐 선입선출(FIFO) 구조
